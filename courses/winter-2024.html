CPS4893 – Machine Learning & Computer Vision

Keywords

Computer Vision, Image Processing, OpenCV, Posture Recognition Application

Course Description

This course provided systematic training in key computer vision techniques and practical application workflows, including image acquisition, processing, feature extraction, and result analysis. I used OpenCV to develop a posture correction prototype that identifies a user's sitting posture in real-time via webcam and provides feedback. The course emphasized hands-on practice and helped me learn how to apply vision technologies to real-world image recognition and data analysis tasks, enhancing my problem-solving skills.
