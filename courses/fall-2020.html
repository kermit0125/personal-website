CPS1231 – Fundamentals of Java Language

Keywords

Java Basic Syntax, Object-Oriented Programming, Program Logic Design

Course Description

This course helped me master the core syntax and logic of the Java programming language, including variables, data types, conditionals, loops, methods, and arrays. I also learned the fundamentals of object-oriented programming, such as classes, objects, and encapsulation. Through small projects, I deepened my understanding and laid a solid foundation for future software development learning.
